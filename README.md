# 🎓 UniQuery – University Query Management System

## 📌 Overview

UniQuery is an AI-powered University Query Management System designed to streamline student support services through intelligent ticket management, AI-based query resolution, and analytics-driven administration.

The system enables students to raise and track tickets, interact with an AI chatbot, search university policies through a knowledge base, and receive automated responses. Administrators and faculty can manage tickets, monitor performance metrics, and generate reports.

## 🤖 AI/ML Features

### Intent Classification

Automatically identifies the category of student queries.

### Priority Prediction

Predicts ticket urgency as Low, Medium, or High.

### Sentiment Analysis

Detects Positive, Neutral, and Negative sentiments.

### Automatic Department Routing

Routes tickets to the correct university department.

### Auto Reply Generation

Generates instant responses for common queries.

### Retrieval-Augmented Generation (RAG)

Answers student questions using university documents such as:

* Academic Calendar
* Student Handbook
* Examination Rules
* Hostel Policies
* Scholarship Guidelines
* Placement Policies

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/your-username/university-query-system.git
cd university-query-system
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Environment

Windows:

```bash
venv\Scripts\activate
```

Linux/Mac:

```bash
source venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Application

```bash
streamlit run app.py
```
